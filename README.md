# Telegram Bot Maker (Telegraf.js + TypeScript + MongoDB)

هذا المشروع عبارة عن نظام متكامل لصناعة بوتات تلجرام متخصصة في إدارة إعلانات القنوات (تبادل إعلاني).

## المميزات
- **صانع البوتات (Main Bot):** يسمح للمستخدمين بإنشاء بوت واحد مجاني.
- **إدارة الإعلانات:** البوتات المصنوعة تقوم بنشر قوائم شفافة (Inline Buttons) في القنوات.
- **نظام المجموعات:** تعيين مجموعة إدارة ومجموعة استقبال لكل بوت مصنوع.
- **إشعارات المسؤول:** إرسال إشعارات لمسؤول النظام عند إنشاء بوت جديد.
- **معمارية قوية:** مبني باستخدام TypeScript و MongoDB وقابل للتوسع.
- **جاهز للنشر:** مهيأ للعمل على منصة Vercel باستخدام Webhooks.

## هيكل المشروع
- `src/core`: الإعدادات الأساسية (قاعدة البيانات، البوت).
- `src/models`: نماذج البيانات (User, Bot, Channel).
- `src/handlers`: معالجات الأوامر والرسائل للبوت الرئيسي والبوتات المصنوعة.
- `src/services`: الخدمات المشتركة (إدارة تشغيل البوتات).

## كيفية التشغيل محلياً
1. قم بتثبيت الحزم: `npm install`
2. قم بإعداد ملف `.env` بناءً على `.env.example`.
3. قم بتشغيل المشروع في وضع التطوير: `npm run dev`

## النشر على Vercel
1. قم برفع المشروع على GitHub.
2. اربط المستودع بمنصة Vercel.
3. أضف المتغيرات البيئية (Environment Variables) في إعدادات Vercel.
4. سيقوم Vercel ببناء ونشر المشروع تلقائياً.

## الأوامر الأساسية
### البوت الرئيسي:
- `/start`: بدء الاستخدام.
- `/create`: إنشاء بوت جديد.
- `/bots`: (للمسؤول) عرض جميع البوتات.

### البوتات المصنوعة:
- `/set_admin`: تعيين مجموعة الإدارة.
- `/set_reception`: تعيين مجموعة الاستقبال.
- `/publish`: نشر القائمة في القنوات.
- `/delete`: حذف القائمة.
